# 刺杀决策 (Assassination) Prompt模板

## 任务描述
好人已经完成了3个任务，现在你可以刺杀梅林。如果成功刺杀梅林，坏人阵营获胜；如果刺杀错误，好人阵营获胜。

## 思考步骤（Chain-of-Thought）

请按照以下步骤进行思考，并在回答中展示你的思考过程：

### 步骤1：回顾游戏历史
仔细分析整个游戏过程：
- 谁在关键时刻给出了关键建议？
- 谁的投票决策总是很准确？
- 谁在发言中表现出很强的洞察力？

### 步骤2：分析梅林的特征
梅林通常有以下特征：
- **有洞察力**：发言中经常分析其他玩家，显得很有洞察力
- **投票准确**：投票决策总是很准确（支持好人队伍，拒绝坏人队伍）
- **引导性强**：在关键时刻给出关键建议
- **隐藏身份**：从不直接指认坏人（因为要隐藏身份）

### 步骤3：排除不可能的人
- 排除你确定的坏人队友
- 排除行为明显不是梅林的玩家

### 步骤4：评估每个候选人
对每个可能的梅林候选人进行评估：
- 发言模式是否符合梅林特征？
- 投票决策是否准确？
- 是否在关键时刻给出关键建议？

### 步骤5：做出最终决策
基于以上分析，选择最可能是梅林的玩家。

## 输出格式

请以JSON格式返回你的决策：

```json
{
    "thinking_process": {
        "step1_game_history": "你的分析...",
        "step2_merlin_characteristics": "你的分析...",
        "step3_exclusion": "你排除了哪些人...",
        "step4_candidate_evaluation": "你对每个候选人的评估...",
        "step5_decision": "你的最终决策理由..."
    },
    "target": 玩家ID
}
```

## 重要提醒
- 这是决定胜负的关键决策
- 仔细分析游戏历史，找出最可能是梅林的玩家
- 你的回答必须基于提供的游戏事实
