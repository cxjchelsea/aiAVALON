# 投票决策 (Vote) Prompt模板

## 任务描述
对提议的队伍进行投票（同意/拒绝）。

## 思考步骤（Chain-of-Thought）

请按照以下步骤进行思考，并在回答中展示你的思考过程：

### 步骤1：检查流局风险（关键！）
- **如果这是第5次投票（vote_round >= 4）**：好人**必须同意**，否则流局会导致坏人直接获胜！
- **如果这是第4次投票（vote_round >= 3）**：好人应该更倾向于同意，避免流局

### 步骤2：分析提议的队伍
- 队伍中包含哪些玩家？
- 这些玩家在你的信念系统中可信度如何？
- 队伍中是否有你确定的坏人（如果你有特殊能力）？

### 步骤3：分析任务历史
- 如果队伍中的玩家之前一起执行过任务，结果如何？
- 是否有可疑的行为模式？

### 步骤4：考虑角色目标
- 如果你是好人：
  - 队伍中有确定的坏人 → 拒绝（除非是第5次投票）
  - 队伍看起来都是好人 → 同意
- 如果你是坏人：
  - 队伍中有坏人队友 → 同意（可以破坏任务）
  - 队伍都是好人 → 拒绝（阻止任务）

### 步骤5：做出决策
基于以上分析，决定是否同意。

## 输出格式

请以JSON格式返回你的决策：

```json
{
    "thinking_process": {
        "step1_rejection_risk": "你的分析...",
        "step2_team_analysis": "你的分析...",
        "step3_mission_history": "你的分析...",
        "step4_role_objectives": "你的分析...",
        "step5_decision": "你的最终决策理由..."
    },
    "vote": true 或 false
}
```

## 重要提醒
- **流局保护**：第5次投票（vote_round >= 4）时，好人必须同意
- 你的回答必须基于提供的游戏事实，不得编造信息
- 如果你是队长，你必须同意自己提议的队伍
