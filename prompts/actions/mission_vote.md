# 任务投票 (Mission Vote) Prompt模板

## 任务描述
作为任务队伍的一员，你需要决定投成功票还是失败票。

## 思考步骤（Chain-of-Thought）

请按照以下步骤进行思考，并在回答中展示你的思考过程：

### 步骤1：检查阵营
- **如果你是好人**：必须投成功票（True）
- **如果你是坏人**：继续以下步骤

### 步骤2：分析当前局势
- 当前是第几轮任务？
- 已经失败了多少个任务？

### 步骤3：分析任务队伍
- 队伍中有哪些玩家？
- 需要几张失败票才能破坏任务？
- 其他坏人队友是否也在队伍中？

### 步骤4：考虑隐藏身份
- 如果已经失败2次，坏人可能已经暴露
- 投成功票可以隐藏身份，但会帮助好人
- 需要平衡破坏任务和隐藏身份

### 步骤5：做出决策
基于以上分析，决定投成功还是失败。

## 输出格式

请以JSON格式返回你的决策：

```json
{
    "thinking_process": {
        "step1_team_check": "你的分析...",
        "step2_current_situation": "你的分析...",
        "step3_mission_team": "你的分析...",
        "step4_identity_hiding": "你的分析...",
        "step5_decision": "你的最终决策理由..."
    },
    "success": true 或 false (true=投成功票, false=投失败票破坏任务)
}
```

## 重要提醒
- 好人必须投成功票
- 坏人需要平衡破坏任务和隐藏身份
- 你的回答必须基于提供的游戏事实
